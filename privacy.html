AI财舵隐私政策

生效日期：2026年7月18日

AI财舵是一款本地优先的个人记账工具，用于识别并整理用户自行选择的本地账单文件。本政策说明当前 Web 试用版如何处理数据。

我们不收集的数据

  • 不要求用户登录银行账户。
  • 不收集银行登录凭证、密码或验证码。
  • 不直接连接银行账户或自动拉取银行数据。
  • 不使用广告 SDK、分析 SDK 或跨应用追踪。
  • 当前公开 Web 版不启用云同步或服务端账本存储。

本地数据处理

用户选择的账单文件会先在当前浏览器中读取。识别出的待确认交易不会自动保存；只有用户点击“确认入账”后，交易记录、分类和用户编辑内容才会保存到浏览器本地存储。

AI 识别

公开 GitHub Pages 版本不能安全保存 AI API 密钥，因此当前版本不在前端内置密钥。未配置安全后端 AI 接口时，应用会尝试使用本地文本解析作为临时降级。未来如接入服务端 AI/OCR，用户主动上传的账单文件或提取出的账单文本可能会发送到该安全服务用于识别，我们会同步更新本政策和应用内说明。

原始文件

当前版本不会长期保存原始账单文件。识别流程完成或用户关闭页面后，原始文件不会作为账本数据保存。

数据删除

用户可以在应用内删除本地数据。删除后，相关账本记录会从浏览器本地存储中移除。
