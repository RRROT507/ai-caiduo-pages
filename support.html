AI财舵支持

个人记账与账单文件识别工具

支持范围

  • 当前 Web 试用版支持上传本地账单文件，并在确认后保存为本地交易记录。
  • 首版优先支持招商银行可复制文字的电子账单内容。
  • 未接入服务端 AI/OCR 时，扫描件、照片和图片型 PDF 可能无法识别。
  • 不支持银行登录、直接连接银行账户或自动拉取银行数据。
  • 识别结果仅供个人整理参考，请以银行官方账单为准。

常见问题

为什么有些 PDF 无法识别？
如果 PDF 本质是图片扫描件，网页无法直接读取其中的文字，需要接入 AI/OCR 服务后才能识别。

识别后会直接保存吗？
不会。账单文件识别后会先显示待确认交易，用户点击“确认入账”后才保存到本地账本。

当前公开版已经接入真正 AI 吗？
公开 GitHub Pages 版本不能安全保存 AI 密钥，因此当前先预留安全后端接口；未配置后端时使用本地解析作为临时降级。
